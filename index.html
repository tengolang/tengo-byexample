Tengo by Example

Tengo is a small, dynamic, fast, secure script language for Go. This site introduces tengolang/tengo v3 through annotated example programs.

Every example runs live in your browser via WebAssembly — no install needed. Click run on any example, or press Ctrl+Enter.

Want to experiment freely? Try the playground.
