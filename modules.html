Modules

← Tengo by Example

Any .tengo file can be a module. Use export to expose a value — typically a map of functions — that other scripts import by filename.

fmt := import("fmt")

math_utils := {
    square: func(x) { return x * x },
    cube:   func(x) { return x * x * x },
    abs:    func(x) { if x < 0 { return -x }; return x },
}

fmt.println(math_utils.square(4))
fmt.println(math_utils.cube(3))
fmt.println(math_utils.abs(-7))

7
